# Customer Segmentation using Clustering

## Author

**Revathi Nukala**

## Project Name

**Customer Segmentation using Clustering**

## Project Objective

The objective of this project is to develop an unsupervised machine learning model that groups customers into different segments based on their purchasing behavior, income, and spending patterns.

Customer segmentation helps businesses understand customer groups, identify high-value customers, and create personalized marketing strategies to improve customer engagement and sales.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- K-Means Clustering
- Jupyter Notebook

## Project Features

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Customer behavior analysis
- Unsupervised machine learning using K-Means clustering
- Customer grouping based on income and spending patterns
- Visualization of customer segments
- Identification of high-value and low-value customer groups
- Cluster analysis for personalized marketing strategies

## Dataset

The project uses customer data containing purchasing behavior and demographic information.

Example features:

- Customer ID
- Gender
- Age
- Annual Income
- Spending Score

The dataset is analyzed to discover hidden customer groups based on similarities in their behavior.

## Project Workflow

1. Import required Python libraries.
2. Load the customer dataset.
3. Perform data cleaning and preprocessing.
4. Analyze customer behavior using Exploratory Data Analysis.
5. Select important features for clustering.
6. Apply K-Means clustering algorithm.
7. Determine optimal number of clusters using the Elbow Method.
8. Assign customers into different segments.
9. Visualize customer clusters.
10. Analyze customer groups for business insights.

## Machine Learning Model

**Algorithm Used: K-Means Clustering**

K-Means is an unsupervised learning algorithm that identifies patterns in customer data and groups similar customers into clusters.

The model segments customers into categories such as:

- High-value customers
- Medium-value customers
- Budget shoppers
- Occasional buyers

## Outcome

The project successfully segments customers based on their income and spending behavior. The generated customer groups help businesses understand customer preferences, improve targeted marketing campaigns, provide personalized offers, and make better business decisions.

The project demonstrates the application of unsupervised machine learning for customer analysis and business intelligence.
